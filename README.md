# 💗 爱心贪吃蛇游戏

一个基于纯HTML/CSS/JavaScript开发的网页版贪吃蛇游戏，提供桌面版和移动版两种体验。

## 📱 版本说明

### 桌面版 (`snake-game.html`)
适合在电脑浏览器上玩，支持键盘操作

### 移动版 (`snake-game-mobile.html`)
**专为苹果手机优化**，支持触屏滑动操作

## ✨ 项目特点

### 共同特点
- 💗 **爱心形状**：蛇和食物都是可爱的爱心形状
- 🎨 **苹果风格设计**：遵循苹果设计语言，优雅简约
- 🎮 **流畅体验**：60fps流畅动画，光晕特效
- 🎵 **音效反馈**：吃食物时有清脆的音效
- 📊 **计分系统**：实时分数和最高分记录（本地存储）
- 🎯 **三种难度**：简单/普通/困难，满足不同水平玩家
- ⏸️ **暂停功能**：随时暂停/继续游戏

### 桌面版特有
- ⌨️ 支持方向键 ↑↓←→ 和 WASD 键控制
- 🖱️ 空格键快速暂停
- 🖥️ 大屏幕 400x400 画布

### 移动版特有
- 👆 **触屏滑动控制**：在画布上滑动即可控制方向
- 🎮 **虚拟方向键**：屏幕底部的方向按钮
- 📱 **全屏适配**：充分利用手机屏幕空间
- 🍎 **iOS优化**：支持添加到主屏幕，全屏模式
- 🔒 **防误触**：禁止页面滚动和缩放
- 📏 **自适应布局**：适配各种iPhone屏幕尺寸

## 🚀 如何运行

### 在电脑上玩
直接在浏览器中打开 `snake-game.html` 文件

### 在手机上玩
1. 在手机浏览器中打开 `snake-game-mobile.html` 文件
2. 建议使用 Safari 浏览器获得最佳体验
3. **推荐**：点击分享按钮 → 添加到主屏幕，可以像App一样使用

## 🎮 游戏操作

### 桌面版
- **移动**: 方向键 ↑↓←→ 或 WASD
- **暂停**: 空格键
- **难度**: 开始前选择难度

### 移动版
- **移动**:
  - 在画布上向任意方向滑动
  - 或点击屏幕下方的方向按钮 ↑↓←→
- **暂停**: 点击"暂停"按钮
- **难度**: 开始前选择难度

## 📖 游戏规则

1. 💗 吃到红色爱心食物可以增长身体并获得 **10分**
2. 💔 撞到墙壁或自己的身体会游戏结束
3. ⚡ 随着分数增加，蛇的移动速度会逐渐提升
4. 🏆 每个难度都有独立的最高分记录

## 🎯 难度说明

| 难度 | 初始速度 | 最快速度 | 提速间隔 | 适合人群 |
|------|---------|---------|---------|---------|
| 🐢 简单 | 慢 | 较慢 | 100分 | 新手玩家 |
| 🚶 普通 | 中等 | 中等 | 50分 | 一般玩家 |
| 🚀 困难 | 快 | 很快 | 30分 | 高手挑战 |

## 🛠️ 技术栈

- HTML5 Canvas（游戏渲染）
- CSS3（毛玻璃效果、渐变、动画）
- Vanilla JavaScript（游戏逻辑）
- Web Audio API（音效生成）
- LocalStorage（最高分存储）

## 📂 文件说明

- `snake-game.html` - 桌面版游戏（键盘控制）
- `snake-game-mobile.html` - 移动版游戏（触屏控制）
- `README.md` - 项目说明文档
- `CLAUDE.md` - 开发规范文档

## 🎨 设计亮点

1. **爱心绘制**：使用贝塞尔曲线精确绘制爱心形状
2. **渐变透明**：蛇身从头到尾逐渐透明，增强立体感
3. **光晕效果**：蛇头和食物都有发光效果
4. **毛玻璃**：移动版使用 backdrop-filter 实现毛玻璃效果
5. **平滑动画**：所有按钮和UI元素都有流畅的过渡动画

## 📱 移动端特别优化

- **全屏沉浸**：支持 iOS 刘海屏的安全区域
- **添加到主屏幕**：可以像原生App一样使用
- **防止缩放**：锁定视口，防止误触缩放
- **触控优化**：增大点击区域，减少误操作
- **性能优化**：移动端使用较小的画布尺寸（320x320）
