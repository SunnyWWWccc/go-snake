# 📱 如何让游戏在手机上运行？

## 问题说明

手机**无法直接打开本地HTML文件**！必须把文件放到网上才能访问。

---

## 🚀 最简单的三种方法

### 方法1️⃣：GitHub Pages（推荐！免费永久）

#### 步骤：

1. **注册GitHub账号**
   - 访问 https://github.com
   - 点击 "Sign up" 注册（免费）

2. **创建仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名输入：`snake-game`
   - 勾选 "Public"（公开）
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 把 `snake-game-mobile.html` 拖进去
   - 点击 "Commit changes"

4. **开启GitHub Pages**
   - 进入仓库的 "Settings"（设置）
   - 左侧点击 "Pages"
   - Source 选择 "main" 分支
   - 点击 "Save"
   - 等待1-2分钟

5. **获取网址**
   - 会显示一个网址，类似：
   - `https://你的用户名.github.io/snake-game/snake-game-mobile.html`
   - 这个网址可以在手机、微信里直接打开！

#### 优点：
- ✅ 完全免费
- ✅ 永久有效
- ✅ 可以在微信里打开
- ✅ 速度快，稳定

---

### 方法2️⃣：Vercel（最快！3分钟）

#### 步骤：

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 用GitHub账号登录（或注册）

2. **创建新项目**
   - 点击 "Add New..." → "Project"
   - 选择 "Import Git Repository"
   - 如果你已经把文件上传到GitHub，选择那个仓库
   - 点击 "Deploy"

3. **自动部署**
   - 等待30秒，自动部署完成
   - 会给你一个网址，类似：`https://你的项目名.vercel.app`

4. **直接访问**
   - 这个网址可以在手机、微信里直接打开！

#### 优点：
- ✅ 超级快速
- ✅ 免费
- ✅ 自动https
- ✅ 全球CDN加速

---

### 方法3️⃣：临时测试（同一WiFi下）

如果你只是想**马上测试一下**，不需要长期使用：

#### 步骤（需要用电脑）：

1. **启动本地服务器**
   - 在项目文件夹里按住 Shift，右键点击空白处
   - 选择"在此处打开命令窗口"或"在终端中打开"
   - 输入以下命令：

   ```bash
   # 如果你有Python（大部分电脑都有）
   python -m http.server 8000

   # 或者如果是Python 2
   python -m SimpleHTTPServer 8000
   ```

2. **查看电脑IP地址**
   - Windows：打开命令提示符，输入 `ipconfig`
   - 找到"IPv4 地址"，比如：`192.168.1.100`

3. **在手机上访问**
   - 确保手机和电脑连接同一个WiFi
   - 在手机浏览器输入：`http://192.168.1.100:8000/snake-game-mobile.html`
   - （把IP地址换成你自己的）

#### 缺点：
- ❌ 只能临时使用
- ❌ 关掉电脑就访问不了
- ❌ 不能在微信里打开（因为不是https）
